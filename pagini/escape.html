Escape HTML

Escaping HTML characters in a string means replacing the:

  • less than symbol (<) with <
  • greater than symbol (>) with >
  • double quotes (") with "
  • single quote (â€™) with '
  • ampersand (&) with &

    • Unescape HTML

Unescaping HTML in a string does the reverse of what we have done above, by replacing:

  • < with <
  • > with >
  • " with "
  • ' with '
  • & with &
