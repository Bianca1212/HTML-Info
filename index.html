HTML: O privire de ansamblu

Ce este HTML?

HyperText Markup Language (HTML) este un limbaj de markup standard utilizat pentru crearea de pagini web. Definește structura și conținutul paginilor web, permițând browserelor web să le afișeze corect. HTML este un limbaj bazat pe text care utilizează etichete pentru a defini diferite elemente ale unei pagini web, cum ar fi titluri, paragrafe, imagini, linkuri și multe altele.

Cum funcționează HTML?

Un fișier HTML este un document text care conține etichete HTML și conținut textual. Etichetele HTML sunt înconjurate de paranteze unghiulare (< >) și descriu tipul de element și pot include atribute care oferă informații suplimentare despre element. De exemplu, eticheta <p> definește un paragraf, iar eticheta <img /> definește o imagine. Browserul web interpretează etichetele HTML și afișează conținutul paginii în consecință.


Tipuri de tag-uri HTML:

Există două tipuri principale de tag-uri HTML:

  1. Tag-uri de container: Acestea definesc un bloc de conținut și au un tag de deschidere și unul de închidere. De exemplu, <p> definește un paragraf de text, iar <table> definește un tabel.
  2. Tag-uri autoînchise: Acestea servesc un scop specific și nu necesită un tag de închidere. De exemplu, <img> definește o imagine, iar <br> introduce o ruptură de linie.

Exemple de etichete HTML comune:

<html>: Definește rădăcina documentului HTML.

  • <!DOCTYPE html>: Declară tipul documentului ca fiind HTML5.
    Această declarație este importantă pentru ca browserele web să interpreteze corect documentul.
  • <html>: Elementul rădăcină al documentului HTML, care conține tot conținutul paginii.
  • ``: Aici se includ toate elementele și secțiunile care definesc structura și conținutul paginii web,
    cum ar fi <head>, <body>, titluri (<h1>, <h2>, etc.), paragrafe (<p>), imagini (<img>), linkuri (<a>), etc.
  • </html>: Marchează sfârșitul documentului HTML.

<head>: Este o componentă esențială a documentului HTML, definind metadate despre pagina web și conținut care nu este afișat direct utilizatorului, dar care este important pentru funcționarea corectă a paginii.

  • <title>: Definește titlul afișat în bara de titlu a browserului și în rezultatele motoarelor de căutare.
  • <meta>: Oferă metadate despre pagina web, cum ar fi descrierea, cuvinte cheie, codificarea caracterelor, etc.
  • <link>: Include resurse externe, cum ar fi foi de stil CSS, fișiere JavaScript, favicon-uri, etc.
  • <script>: Include cod JavaScript care este executat pe partea clientului.
  • <base>: Definește adresa URL de bază pentru linkurile relative din pagina web.

<body>: Este o componentă esențială a documentului HTML, definind conținutul principal al paginii web care este vizibil utilizatorului.

  • Tag-ul <body> trebuie să conțină conținutul principal al paginii web care este vizibil utilizatorului.
  • Puteți organiza conținutul folosind secțiuni de structură (<header>, <nav>, <main>, etc.) și elemente de formatare a textului.
  • Integrați elemente multimedia (<img>, <video>, <audio>) pentru a îmbogăți pagina web.
  • Creați formulare (<form>) pentru a colecta date de la utilizatori.
  • Folosiți linkuri (<a>) pentru a conecta diferite pagini web sau resurse externe.
  • Structurați listele (<ul>, <ol>, <dl>) și tabelele (<table>, <tr>, <th>, <td>) pentru a prezenta informații organizate

<h1> până la <h6>:Tag-urile <h1> până la <h6> (numite și heading tags) sunt utilizate în HTML pentru a defini titlurile și subtitlurile din cadrul unei pagini web. Ele creează o ierarhie vizuală a conținutului, ajutând utilizatorii să scaneze rapid pagina și să înțeleagă structura informației.

  • <h1> definește titlul principal al paginii, cel mai important titlu.
  • <h2> definește un subtitlu important, subordonat titlului principal.
  • <h3> definește un subtitlu mai puțin important, subordonat unui titlu <h2>.
  • <h4>, <h5>, <h6> definesc subtitluri din ce în ce mai puțin importante, urmând o ierarhie descendentă.

<a> Tag-ul <a> este fundamental în HTML pentru a crea hyperlink-uri, permițând utilizatorilor să navigheze între pagini web sau să acceseze resurse externe.

  • href este un atribut obligatoriu care specifică adresa URL a destinației linkului.
  • text-care-se-afișează este textul pe care utilizatorul îl va vedea și pe care va putea face clic pentru a accesa linkul.

<p>Tag-ul <p> (paragraf) este un element fundamental în HTML, utilizat pentru a defini paragrafe de text în cadrul unei pagini web. Fiecare paragraf este formatat separat, cu spațiu vertical deasupra și dedesubt, oferind o structură clară conținutului textual.

  • Tag-ul <p> nu necesită un element de închidere.
    Un paragraf se termină automat la întâlnirea unui alt tag sau a sfârșitului de linie.
  • Puteți utiliza tag-uri HTML în interiorul unui paragraf pentru a formata textul, a crea linkuri sau a afișa cod.
  • Este recomandat să folosiți paragrafe scurte și concise pentru a îmbunătăți lizibilitatea textului.
  • Puteți personaliza aspectul paragrafelor cu CSS, definind marginile, spațierea, alinierea și alte caracteristici.

<img>Tag-ul <img> este esențial în HTML pentru a integra imagini în paginile web. Permite afișarea de fotografii, ilustrații, logo-uri și alte elemente grafice, îmbogățind conținutul vizual și sporind atractivitatea paginii.

  • src este un atribut obligatoriu care specifică adresa URL a fișierului imagine.
  • alt este un atribut important care definește un text alternativ,
    afișat în cazul în care imaginea nu poate fi încărcată sau utilizatorul are activat un cititor de ecran.
  • width și height sunt atribute opționale care stabilesc lățimea și înălțimea imaginii în pixeli.

<table>Tag-ul <table> este fundamental în HTML pentru a crea tabele care structurează și organizează conținutul textual într-un mod clar și concis. Permite afișarea datelor tabulare, cum ar fi liste, statistici, comparații și alte informații complexe, într-un format ușor de scanat și înțeles.

  • <table>: Definește începutul tabelului.
  • <tr>: Definește un rând de date din tabel.
  • <th>: Definește o celulă de tip antet, de obicei afișată cu o formatare mai distinctă (bold, italic).
  • <td>: Definește o celulă de date obișnuită.

Atributele tag-urilor HTML:

Atributele tag-urilor HTML oferă informații suplimentare despre elementul respectiv. Ele sunt definite ca perechi nume-valoare în interiorul tag-ului de deschidere. De exemplu, eticheta <img> poate avea un atribut src care specifică URL-ul imaginii, iar eticheta <a> poate avea un atribut href care specifică URL-ul hyperlink-ului.

Exemple de atribute HTML comune:

  • src: Specifică URL-ul unei resurse, cum ar fi o imagine sau un script.
  • href: Specifică URL-ul unui hyperlink.
  • alt: Oferă un text alternativ pentru o imagine, care este afișat atunci când imaginea nu se poate încărca sau când utilizatorul utilizează un cititor de ecran.
  • id: Definește un identificator unic pentru un element.
  • class: Definește o clasă CSS pentru un element.
  • style: Definește stiluri CSS inline pentru un element.

Beneficiile utilizării HTML:

  • Simplu și ușor de învățat: Sintaxa HTML este relativ simplă și ușor de înțeles, făcând-o accesibilă persoanelor cu diferite niveluri de experiență.
  • Versatil: HTML poate fi utilizat pentru a crea o gamă largă de pagini web, de la site-uri web simple la aplicații web complexe.
  • Fundal: Setați o imagine de fundal pentru elemente. Decorare: Controlați aspectul marginilor, liniilor și hyperlinkurilor.
  • Standard: HTML este un limbaj standardizat, asigurând compatibilitatea cu toate browserele web majore.
  • Baza pentru web: HTML este limbajul fundamental al World Wide Web și este esențial pentru crearea oricărei pagini web.
